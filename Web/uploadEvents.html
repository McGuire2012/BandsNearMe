Book performances

You can either upload a file or use the textbox provided to create a single show or multiple shows at once! Follow this format to create an event.

"BandName, Venue Name, Description of show, Show date (MM/DD/YYYY), Show time (Millitary 20:00 or 8:00pm)". This will guarentee the show to go through.

No special characters cannot be used at this time (' , : ; \ / { } etc...). Otherwise the request will not be processed.

To create multiple shows at once, press enter to start a new line and follow the templete again.
